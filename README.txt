Allow WooCommerce sites to be published statically.

== Description ==

Uses Snipcart to allow WooCommerce checkout functions to work on static sites.

== Installation ==

Upload the ZIP to your WordPress plugins page within your dashboard.

Activating the plugin will add a new Snipcart submenu to the WP2Static main menu.
